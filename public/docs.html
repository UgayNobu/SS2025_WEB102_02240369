API Documentation

Welcome to the API documentation page.

Base URL

https://yourapi.com/api

Endpoints

GET /users

Retrieves a list of users.

curl -X GET https://yourapi.com/api/users

POST /users

Creates a new user.

curl -X POST https://yourapi.com/api/users -d '{"name": "John Doe"}' -H "Content-Type: application/json"

GET /users/:id

Retrieves a specific user by ID.

curl -X GET https://yourapi.com/api/users/1

PUT /users/:id

Updates a user's information.

curl -X PUT https://yourapi.com/api/users/1 -d '{"name": "Jane Doe"}' -H "Content-Type: application/json"

DELETE /users/:id

Deletes a user.

curl -X DELETE https://yourapi.com/api/users/1
